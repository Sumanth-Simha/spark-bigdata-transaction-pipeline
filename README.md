# Spark Big Data Transaction Pipeline

This repository demonstrates the use of Apache Spark for processing and analyzing transaction datasets. The current implementation includes exploratory data analysis, fraud analytics, visualizations, and a baseline fraud detection model using Logistic Regression.

Current Status:

* Data ingestion using PySpark
* Exploratory Data Analysis (EDA)
* Fraud distribution analysis
* Time-based fraud analysis
* Correlation analysis
* Visualizations
* Logistic Regression using Spark MLlib

Future Enhancements:

* CSV to Parquet conversion
* Partitioned storage
* Spark SQL analytics
* Caching optimization
* Streaming concepts
* End-to-end Big Data pipeline architecture
